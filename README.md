# jarvis-mark-80
J.A.R.V.I.S. (Just A Rather Very Intelligent System)  A cinematic, full-stack AI assistant application featuring a custom-built Marvel HUD interface. Powered by Google’s Gemini 2.5 Flash model, it integrates multimodal input (voice and text) with real-time text-to-speech (TTS) output, providing a futuristic and interactive user experience.
